codejam
=======

C&amp;UMS(Content and Users Management System) for hosting small coding contests

Features:

Front End:
   User Side:
      - Registration
      - Solution Submission
      - Activity Feedback      
      - Notifications and Competetion Schedules
      - Leaderboard      
    Admin Side:
      - Participants Manager
      - Problem Submission
      - Scoring Manager

Back End:

      - Code Testing Framework (C,C++,JAVA,PYTHON... etc )
